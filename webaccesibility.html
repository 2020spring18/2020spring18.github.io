What is Web Accessibility

Web Accesibility is the idea of creating easy to navigate websites, and a great deal of it relates to considering factors such as age or disease that might hinder online use. The WCAG (Web Content Accesibility Guide) is a prominent guideline on the internet for web accesibility created by the W3C. For more information, see the link below

https://www.w3.org/WAI/standards-guidelines/wcag/

Section 508 Comlpiance is a federal law that requires all federal agencies to ensure that their website is accessible to all people.

Website Evaluation Tools

The following are excellent tools used by website developers for the purpose of increasing web accesibility on their site

1. WAVE is a web accesibility evaluation tool created by Web-Aim. WAVE has extenions on both Google Chrome and Firefox, allowing for quick and easy evaluation of a website
2. Cynthia Says Portal is an online evaluation tool that allows you to check if your website is in compliance to either the WCAG or Section 508
3. AY11 Compliance Platform is a web accessibility evaluation tool that can be used to ensure that your website is up to date on the guidelines and standards set by the WCAG, Section 508, and Americans With Disabilities.

Demonstrational Simulations

Online simulations allow people to see what it is like to have a condition that could hinder the online experience.

*Below are a list of websites that show the experience that disabled individuals have while browsing online:
1. Color Labrotory - emulates how individuals with colorblindess see the world.
2. Colorblind Web Page Filter - online web filter that allows for a person to see the colors on a website as a colorblind individual would.
3. JAWS - screen reader that allows for those suffering from vision loss to be able to understand the content on a website. Is it avaliable for download on Windows 10.
